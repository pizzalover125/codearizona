# codearizona

Code Arizona is Arizona's premier high school coding competition. Code Arizona is hosted by The Code Hatch, a nonprofit dedicated to hosting hackathons and STEM events for teenagers in Phoenix! This is the source code of the website! 

### Techincal Information

The website was made in HTML, CSS, and JavaScript. A framework was not used because the site is one page and static. The website can be viewed at https://codearizona.netlify.app/.
